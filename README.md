# Hemicycle

A web app for following voting activity in the French National Assembly (17th legislature). It ingests official open data on public votes from the Assemblée nationale, makes them easy to browse and filter — by date, theme, or parliamentary group — and adds AI-generated, source-grounded explanations and session summaries for the most significant votes.
